Kiến Thức Cơ Bản Về Java – Bắt Đầu Với Những Khái Niệm Đơn Giản

Java là một ngôn ngữ lập trình hướng đối tượng mạnh mẽ và phổ biến, được phát triển bởi Sun Microsystems vào năm 1995. Bài viết này sẽ giúp bạn hiểu về Java từ những khái niệm cơ bản nhất như cấu trúc chương trình, các kiểu dữ liệu cơ bản, câu lệnh điều kiện và vòng lặp.

1) Giới thiệu về Java

Java là một ngôn ngữ lập trình hướng đối tượng, độc lập nền tảng, nghĩa là bạn có thể viết mã Java một lần và chạy ở bất kỳ hệ điều hành nào hỗ trợ Java. Java sử dụng **JVM (Java Virtual Machine)** để thực thi mã. Đây là điểm khác biệt lớn so với nhiều ngôn ngữ khác.

2) Cấu trúc cơ bản của một chương trình Java

Chương trình Java cơ bản gồm ba phần chính: Lớp, Phương thức, và Dòng lệnh. Cấu trúc đơn giản nhất của một chương trình Java trông như sau:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}

- **public class HelloWorld**: Đây là khai báo lớp của chương trình.
- **public static void main(String[] args)**: Phương thức main, nơi chương trình bắt đầu thực thi.
- **System.out.println("Hello, Java!");**: In ra màn hình dòng chữ "Hello, Java!".

3) Các kiểu dữ liệu cơ bản trong Java

Java có 8 kiểu dữ liệu cơ bản: `byte`, `short`, `int`, `long`, `float`, `double`, `char`, và `boolean`. Dưới đây là một số ví dụ:

int age = 25;
float price = 19.99f;
char grade = 'A';
boolean isJavaFun = true;

4) Câu lệnh điều kiện và vòng lặp

Câu lệnh điều kiện giúp kiểm tra một biểu thức và thực hiện hành động dựa trên kết quả. Ví dụ:

if (age > 18) {
    System.out.println("Bạn là người trưởng thành.");
} else {
    System.out.println("Bạn chưa đủ tuổi.");
}

Vòng lặp `for` dùng để lặp qua một đoạn mã nhiều lần:

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

5) Lớp và đối tượng trong Java

Trong Java, lớp (class) là một khuôn mẫu dùng để tạo đối tượng. Mỗi đối tượng có các thuộc tính (fields) và phương thức (methods) riêng. Ví dụ:

public class Car {
    String color;
    int year;

    public void drive() {
        System.out.println("The car is driving.");
    }
}

Để sử dụng lớp này, bạn tạo đối tượng (object) từ lớp `Car`:

Car myCar = new Car();
myCar.color = "Red";
myCar.year = 2020;
myCar.drive();
