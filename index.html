[01] Giới thiệu về Java

Giới thiệu cơ bản về Java và các mốc lịch sử quan trọng của Java.

01
Cú pháp nền tảng Java

Khám phá cú pháp cơ bản của Java như khai báo biến, kiểu dữ liệu, toán tử và các cấu trúc điều kiện như if/switch.

02
Lập trình hướng đối tượng (OOP)

Các khái niệm cơ bản về lập trình hướng đối tượng như Class, Object, Kế thừa, Đa hình, Đóng gói.

03
Xử lý ngoại lệ trong Java

Sử dụng try-catch-finally để bắt lỗi, cùng với throw/throws để ném và xử lý ngoại lệ.

04
Java 8: Lambda & Stream

Khám phá phong cách lập trình hàm (functional style), sử dụng Stream API, Optional và Date/Time API trong Java 8.

05
Giới thiệu về JavaScript

Khám phá cú pháp cơ bản của JavaScript, bao gồm biến, hàm, DOM, và thao tác với API bằng fetch.

06
Kiểm thử trong Java

JUnit 5, Mockito, Test Pyramid, viết Unit/Integration test, cấu hình Maven/Gradle và coverage report.

07
Bảo mật Java

Cấu trúc của một bài viết chi tiết, với các thành phần hay dùng và cách tổ chức thông tin.

08
WebSocket trong Java

Giới thiệu về WebSocket trong Java, sử dụng để truyền tải dữ liệu theo thời gian thực trong các ứng dụng web.

09

  • Giai đoạn 2: Ứng dụng & Dữ liệu

    Xây project hoàn chỉnh. Java → Spring Boot REST API; JS → React/Node. Deploy GitHub Pages/Vercel.
