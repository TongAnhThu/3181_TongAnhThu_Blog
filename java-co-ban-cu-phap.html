Java Cơ Bản - Cú Pháp (Syntax)

Java là một ngôn ngữ lập trình hướng đối tượng mạnh mẽ và phổ biến, với cú pháp dễ hiểu và dễ làm quen. Bài viết này sẽ cung cấp cho bạn cái nhìn tổng quan về cú pháp cơ bản trong Java, bao gồm cách khai báo biến, sử dụng kiểu dữ liệu, câu lệnh điều kiện, vòng lặp và các khái niệm lập trình quan trọng khác.

Nội dung
  • 1) Khai báo biến và kiểu dữ liệu trong Java
  • 2) Câu lệnh điều kiện (if, switch)
  • 3) Vòng lặp (for, while)
  • 4) Lớp và đối tượng trong Java

1) Khai báo biến và kiểu dữ liệu trong Java

Để khai báo một biến trong Java, ta sử dụng cú pháp: kiểu_dữ_liệu tên_biến;. Java có các kiểu dữ liệu cơ bản như sau:

  • byte: 8-bit, dùng để lưu trữ các số nguyên nhỏ
  • int: 32-bit, dùng để lưu trữ số nguyên thông dụng
  • double: 64-bit, dùng để lưu trữ số thực
  • boolean: Dùng để lưu trữ giá trị true hoặc false
  • char: Dùng để lưu trữ ký tự Unicode

Ví dụ khai báo một số biến trong Java: 

int number = 10;
double price = 99.99;
boolean isJavaFun = true;
char grade = 'A';

2) Câu lệnh điều kiện (if, switch)

Câu lệnh điều kiện cho phép chương trình kiểm tra các điều kiện và quyết định hành động cần thực hiện.

  • if: Kiểm tra một điều kiện và thực thi khối mã nếu điều kiện đó đúng
  • switch: Kiểm tra nhiều trường hợp có thể xảy ra, thích hợp khi có nhiều điều kiện với giá trị rõ ràng.

Ví dụ sử dụng câu lệnh if:

int number = 10;
if (number > 5) {
    System.out.println("Số lớn hơn 5");
} else {
    System.out.println("Số nhỏ hơn hoặc bằng 5");
}

Ví dụ sử dụng câu lệnh switch:

int day = 3;
switch (day) {
    case 1:
        System.out.println("Thứ hai");
        break;
    case 2:
        System.out.println("Thứ ba");
        break;
    case 3:
        System.out.println("Thứ tư");
        break;
    default:
        System.out.println("Ngày không hợp lệ");
}

3) Vòng lặp (for, while)

Vòng lặp giúp thực hiện một khối mã nhiều lần trong chương trình.

  • for: Dùng khi biết trước số lần lặp
  • while: Dùng khi không biết trước số lần lặp, vòng lặp sẽ tiếp tục cho đến khi điều kiện không còn đúng

Ví dụ sử dụng vòng lặp for:

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

Ví dụ sử dụng vòng lặp while:

int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}

4) Lớp và đối tượng trong Java

Trong Java, lớp (class) là một khuôn mẫu để tạo ra đối tượng. Lớp định nghĩa các thuộc tính và phương thức mà đối tượng sẽ có.

  • Lớp (Class): Là một bản thiết kế cho đối tượng, chứa các thuộc tính và phương thức.
  • Đối tượng (Object): Là một thể hiện cụ thể của lớp.

Ví dụ về lớp và đối tượng trong Java:

class Car {
    String color;
    int speed;

    void accelerate() {
        speed += 10;
    }
}

public class Main {
    public static void main(String[] args) {
        Car myCar = new Car();  // Tạo đối tượng từ lớp Car
        myCar.color = "Red";
        myCar.speed = 0;
        myCar.accelerate();  // Gọi phương thức của đối tượng
        System.out.println("Speed: " + myCar.speed);
    }
}
