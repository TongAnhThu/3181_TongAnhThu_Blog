Java 8: Những Tính Năng Mới Quan Trọng

Java 8 mang đến bước nhảy lớn cho lập trình Java: biểu thức Lambda, Stream API, phương thức mặc định trong interface, Optional để xử lý null an toàn, và API ngày–giờ hoàn toàn mới trong gói java.time.

Nội dung
  • 1) Lambda Expressions
  • 2) Stream API
  • 3) Default Methods
  • 4) Optional
  • 5) Date & Time API

1) Lambda Expressions

Lambda giúp viết hàm ngắn gọn, truyền hành vi như một giá trị và tận dụng các API chức năng.

import java.util.Arrays;
import java.util.List;

public class LambdaDemo {
    public static void main(String[] args) {
        List<String> names = Arrays.asList("John", "Jane", "Jack");

        // Lambda:
        names.forEach(name -> System.out.println(name));

        // Method reference (tương đương, ngắn hơn):
        names.forEach(System.out::println);
    }
}

2) Stream API

Xử lý dữ liệu theo kiểu hàm: lọc, biến đổi, gom nhóm… dễ đọc và dễ song song hóa.

import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

public class StreamDemo {
    public static void main(String[] args) {
        List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5, 6);

        List<Integer> evenSquares = numbers.stream()
                .filter(n -> n % 2 == 0)   // lọc số chẵn
                .map(n -> n * n)           // bình phương
                .collect(Collectors.toList());

        System.out.println(evenSquares);   // [4, 16, 36]
    }
}

Mẹo: Dùng stream().parallel() cẩn thận — đo trước tối ưu sau.

3) Default Methods

Interface có thể cung cấp triển khai mặc định, giúp mở rộng API mà không phá vỡ lớp đang dùng.

interface Greeter {
    default void greet() {
        System.out.println("Hello from default method!");
    }
}

class ConsoleGreeter implements Greeter { }

public class DefaultMethodDemo {
    public static void main(String[] args) {
        new ConsoleGreeter().greet(); // dùng ngay triển khai mặc định
    }
}

4) Optional

Optional<T> giúp diễn đạt “có thể không có giá trị”, giảm NullPointerException bằng API tường minh.

import java.util.Optional;

public class OptionalDemo {
    static String getNameOrNull() {
        return Math.random() > 0.5 ? "Alice" : null;
    }

    public static void main(String[] args) {
        Optional<String> name = Optional.ofNullable(getNameOrNull());

        // Chỉ in nếu có giá trị
        name.ifPresent(System.out::println);

        // Giá trị mặc định khi rỗng
        String safe = name.orElse("Unknown");

        // Ném ngoại lệ tùy ý khi rỗng
        // String must = name.orElseThrow(() -> new IllegalStateException("No name"));
    }
}

Tránh lạm dụng Optional.get(); ưu tiên orElse, orElseGet, orElseThrow, ifPresent.

5) New Date & Time API (java.time)

API ngày–giờ mới (JSR-310) bất biến, rõ ràng và an toàn luồng, thay thế java.util.Date/Calendar. 

import java.time.*;

public class TimeDemo {
    public static void main(String[] args) {
        LocalDate today = LocalDate.now();
        LocalDate nextWeek = today.plusWeeks(1);

        LocalDate birth = LocalDate.of(2000, 10, 1);
        Period age = Period.between(birth, today);

        ZonedDateTime nowHanoi = ZonedDateTime.now(ZoneId.of("Asia/Ho_Chi_Minh"));

        System.out.println(today + " → " + nextWeek);
        System.out.println("Age: " + age.getYears() + " years");
        System.out.println("Hanoi: " + nowHanoi);
    }
}

Dùng DateTimeFormatter để định dạng/parse; ưu tiên ZonedDateTime khi cần múi giờ.
